
# Web-Based CMS (Spring 2026)
### 🤠 Austin Community College

Welcome to the class repository! This is the "Golden Master" that you will fork to create your own portfolio.

## 🛠 Status
* **Maintainer:** Professor Watkins
* **System:** Docs-as-Code (Markdown + Confluence)
* **License:** MIT

## 🚀 Getting Started
1. Click the **Fork** button (top right) to copy this to your account.
2. Click the green **Code** button -> **Codespaces** -> **Create codespace on main**.
3. Wait for the terminal to load.

## 📂 Repository Structure
* `.devcontainer/` - Configuration for your cloud computer.
* `.github/` - The automation workflow that publishes to Confluence.
* `docs/` - The actual documentation content lives here.
* `legacy-content/` - The messy "source files" you will fix in Project 1.
